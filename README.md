# WP Simple Events

WP Simple Events is a lightweight WordPress plugin to create and display events using a custom post type and shortcode.

## Features
- Custom Post Type: Event
- Event Date meta field
- Secure data handling (nonces, sanitization)
- Shortcode to display events

## Installation
1. Download or clone this repository.
2. Upload the `wp-simple-events` folder to `/wp-content/plugins/`.
3. Activate the plugin from the WordPress admin panel.
